# Professional Portfolio

A high-end, single-page portfolio built with **React**, **Tailwind CSS**, and **Framer Motion**. Deep midnight theme with glassmorphism, electric indigo & cyan accents.

## Features

- **Hero**: Typewriter roles, magnetic CTA, Live Coding Terminal (JSON bio)
- **Project Gallery**: Glassmorphic cards with hover scale, GitHub & Live Demo links, tech badges
- **Skills Cloud**: Staggered fade-in on scroll
- **Experience Timeline**: Vertical line with glowing nodes
- **Testimonial Carousel**: Social proof section
- **Contact Form**: Glass-style form

## Run

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
npm run preview
```

## Customize

- **Content**: Edit `src/data/projects.js`, `src/data/skills.js`, `src/data/experience.js`, `src/data/testimonials.js`, and the JSON in `TerminalBio.jsx`.
- **Theme**: Base `#030712`, primary `#6366f1`, accent cyan in `src/index.css` and components.
